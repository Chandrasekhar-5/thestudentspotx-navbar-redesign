# The Student Spot

A student-focused platform designed to connect students, startups, companies, colleges, and coaching institutions through opportunities, resources, and community-driven initiatives.

## Features
- Responsive modern UI
- Mobile-friendly animated menu
- Role-based pages (Students, Startups, Companies, Colleges, Coaching)
- Grid-based layouts for scalability
- Clean hover & transition effects

## Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript

## Responsive Design
- Optimized for mobile, tablet, and desktop
- Adaptive layouts using CSS Grid
- Smooth mobile navigation with scroll lock
